# Sniffing Skill

Raycast extension for searching, reading, editing, and organizing local AI skills without opening a separate desktop app.

## Commands

- `Search Skills`: scan common local skill folders, search by title and description, preview full content, edit in place, open files, and copy/install a skill into another tool folder.

## Supported scan folders

- Claude: `~/.claude/skills`, `~/.agents/skills`
- Codex: `~/.codex/skills`
- Cursor: `~/.cursor/skills`, `~/.cursor/rules`
- Windsurf: `~/.codeium/windsurf/memories`, `~/.windsurf/rules`
- Amp: `~/.config/amp`
